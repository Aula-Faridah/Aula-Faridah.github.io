APA ITU BOOTSTRAP?


source : tutorialrepublic.com


Bootstrap adalah framework untuk membuat responsif website menggunakan HTML, CSS, and JS. Bootstrap menyediakan class-class CSS yang sudah terintegrasi dengan javascript dan jQuery. sehingga kita cukup memanggil class-class yang sudah disediakan ketika ingin menciptakan website yang memiliki desain menarik.

Bootstrap memudahkan penggunanya untuk membuat tata letak web yang fleksibel dan responsif. Bootstrap sudah mencakup template desain untuk membuat komponen antarmuka pengguna yang umum seperti formulir, tombol, navigasi, dropdown, peringatan, modals, tab, akordeon, caousel, tooltips, dan lain sebagainya.

Apa Saja Keuntungan Menggunakan Bootstrap?

Adapun keuntungan menggunakan Bootstrap untuk mengembangkan web antara lain :

  1. Menghemat banyak waktu
  2. Memiliki fitur responsif
  3. Desainnya konsisten
  4. Mudah digunakan
  5. Kompatibel dengan berbagai browser
  6. Open Source
